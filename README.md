# LUPE 2.0 Analysis Package
LUPE-ASOiD powered pipeline with behavior analysis scripts.


### In a virtual env
Preferably in Python 3.11

```commandline
pip install -r requirements.txt 
```

### If no local IDE (not preferred)
```commandline
jupyter lab
```
